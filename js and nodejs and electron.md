# 环境安装

### windows

首先要安装nodejs，我在windows安装的。nodejs安装好后就有npm，我是在windows下安装的，nodejs直接装应用程序就行了。

安装electron

```bash
npm i -D electron@latest
```

demo环境

```bash
git clone https://github.com/electron/electron-api-demos
cd electron-api-demos
npm install
npm start
```



### WSL

```bash
sudo apt-get install -y nodejs
git clone https://github.com/electron/electron-api-demos
cd electron-api-demos
npm install
npm start
```
